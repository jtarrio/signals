# Signals

Signals is a TypeScript library that lets you demodulate radio signals in real time from a JavaScript or TypeScript application.

You can use Signals to write a radio receiver application with stereo FM demodulation in less than 10 lines of JavaScript.

## Installing

You can install Signals using NPM:

```shell
npm install @jtarrio/signals
```

## Using Signals

- Demodulating and playing signals in real time through the [Radio API](radio.md)
- Writing a [new signal source](new-source.md)
- Writing a [new modulation scheme](new-demod.md)
- Using the [built-in DSP library](dsp.md)
