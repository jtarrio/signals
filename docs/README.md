# _Demodulator_

_Demodulator_ is a TypeScript library that lets you demodulate radio signals in real time from a JavaScript or TypeScript application.

You can use _Demodulator_ to write a stereo FM demodulator in less than 10 lines of JavaScript.

## Installing

You can install _Demodulator_ using NPM:

```shell
npm install @jtarrio/demodulator
```

## Using _Demodulator_

- Creating a radio demodulator through the [Radio API](radio.md)
- Writing a [new modulation scheme](new-demod.md) for the Radio API
- Using the [built-in DSP library](dsp.md)
